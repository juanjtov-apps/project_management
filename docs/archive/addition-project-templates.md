# Addition Project Templates - Scenario-Based Reference

This document provides structured templates for common addition project scenarios, including detailed stage breakdowns and finish materials. Each scenario represents a typical addition configuration with specific room combinations and stage counts.

---

## Quick Reference Table

| Scenario | Rooms | Typical Size | Total Stages |
|----------|-------|--------------|--------------|
| Master Suite Addition | 1 master bedroom, 1 master bathroom, walk-in closet | 400-600 sf | 18 |
| Guest Suite Addition | 1 guest bedroom, 1 guest bathroom, 1 walk-in closet | 200-350 sf | 18 |
| Multi-Bedroom Addition | 2 bedrooms, 1 shared bathroom, hallway | 500-700 sf | 18 |
| Family Wing Addition | 2 bedrooms, 2 bathrooms, hallway | 700-1000 sf | 24 |
| Kitchen Extension | Kitchen area, dining area, optional pantry | 200-400 sf | 18 |

## Scenario 1: Master Suite Addition (400-600 sf)

### Description

A master suite addition expands the home with a new master bedroom, master bathroom with walk-in shower, and integrated walk-in closet. This configuration is ideal for homeowners seeking a private retreat with upgraded amenities.

### Rooms Included

- Master Bedroom (~250-350 sf)
- Master Bathroom with Walk-in Shower (~100-150 sf)
- Walk-in Closet (integrated into bedroom layout)

### Stages

**SHARED STAGES (1-12)**

**1. Drawings/Design**
- DESIGN AND DRAFTING OF EXISTING AND PROPOSED FLOOR PLANS AND ELEVATIONS (AS APPLICABLE AND NECESSARY FOR PERMIT)
- PREPARATION OF A STAMPED SET OF STRUCTURAL CALCULATIONS AND DETAILED DRAWINGS FOR AN ADDITION BASED ON AN ON-SITE VISIT TO GATHER SITE CONDITIONS BASED ON VISUALLY OBSERVABLE CONDITIONS
- ANALYSIS FOR WIND, SEISMIC AND GRAVITY FORCES ACCORDING TO THE CURRENT CALIFORNIA BUILDING CODE
- TITLE 24 ENERGY CALCULATION SERVICES
- SUBMIT TO THE CITY FULL SET OF BLUEPRINTS PRINTED FOR FEW COPIES
- REVISE ANY CHANGES MADE BY THE CITY AND RESUBMIT UNTIL APPROVAL

**2. Demolition**
- PROVIDE DUMPSTERS AS NEEDED
- COVER ALL WORK AREA AS NEEDED PRIOR TO DEMOLITION
- DEMOLISH THE EXTERIOR WALL WHERE ADDITION CONNECTS TO EXISTING HOUSE
- DEMOLISH DECK/PATIO/PORCH AS NEEDED
- CLEAN UP AND HAUL ALL DEBRIS

**3. Foundations**
- DIG TO ACCOMMODATE CONCRETE FOUNDATIONS PER APPROVED PLANS
- INSTALL REBAR PER APPROVED PLANS
- POUR CONCRETE FOR FOUNDATIONS PER APPROVED PLANS
- INSTALL HOLD DOWNS AND EPOXY PER APPROVED PLANS
- INSTALL ANCHOR BOLTS PER APPROVED PLANS
- ANY ADDITIONAL FOUNDATION DETAILS WILL BE DONE PER APPROVED PLANS

**4. Framing**
- INSTALL FRAMING FOR APPX: 400-600 SQ-FT ADDITION: (1) MASTER BEDROOM, (1) MASTER BATHROOM, (1) WALK-IN CLOSET
  - INSTALL FRAMING TO EXTEND THE EXISTING EXTERIOR WALLS OF THE HOUSE TO ACCOMMODATE THE ADDITIONAL SQUARE FOOTAGE PER APPROVED PLANS
  - CONNECT THE NEW WALLS TO THE EXISTING STRUCTURE, ENSURING PROPER ALIGNMENT AND ATTACHMENT FOR STRUCTURAL INTEGRITY PER APPROVED PLANS
  - FRAME THE INTERIOR WALLS OF THE ADDITION ACCORDING TO THE ARCHITECTURAL PLANS AND DESIRED LAYOUT PER APPROVED PLANS
  - INSTALL ANY NECESSARY WOOD LOAD-BEARING ELEMENTS TO SUPPORT THE NEW STRUCTURE PER APPROVED PLANS
  - INSTALL FRAMING FOR THE NEW FLOOR, CEILING AND ROOF OF THE ADDITION PER APPROVED PLANS
  - INSTALL FRAMING FOR THE NEW WINDOWS AND DOORS ON THE ADDITION PER APPROVED PLANS
- INSTALL FRAMING FOR NEW MASTER BATHROOM LAYOUT PER APPROVED PLANS
- ANY ADDITIONAL FRAMING DETAILS WILL BE DONE PER APPROVED PLANS

**5. Roof**
- INSTALL PLYWOOD FOR ROOFING IN THE ADDITION PER APPROVED PLANS
- INSTALL WATERPROOF TAR PAPER ON THE NEW ROOF PER APPROVED PLANS
- INSTALL METAL FLASHING DRIP EDGE AND AROUND PIPES AND VENTS
- INSTALL HD SHINGLES WITH UV PROTECTION FOR EXTRA ENERGY EFFICIENCY ON THE ADDITION TO MATCH (SHINGLES ARE INCLUDED)
- INSTALL RAIN GUTTERS IN THE NEW ROOF
- INSTALL DOWNSPOUTS

**6. Windows and Doors**
- WATERPROOFING PRIOR WINDOWS/DOORS INSTALLATION WITH FLASHING PAPER PER APPROVED PLANS
- INSTALL NEW WINDOWS AND DOORS THROUGHOUT THE ADDITION PER APPROVED PLANS
- ANY ADDITIONAL DETAILS WILL BE DONE PER APPROVED PLANS

**7. Electrical (Rough)**
- INSTALL NEW OUTLETS AND SWITCHES IN (1) MASTER BATHROOM-TAMPER RESISTANT GFCI OUTLETS PER APPROVED PLANS
- INSTALL NEW OUTLETS AND SWITCHES THROUGHOUT THE ADDITION PER APPROVED PLANS
- INSTALL RECESSED LIGHTS THROUGHOUT THE ADDITION PER APPROVED PLANS
- INSTALL NEW HARD LINE TO PANEL, IN ORDER TO INSTALL NEW EXHAUST FAN FOR (1) MASTER BATHROOM PER APPROVED PLANS
- ANY ADDITIONAL ELECTRICAL DETAILS WILL BE DONE PER APPROVED PLANS

**8. Plumbing (Rough)**
- INSTALL NEW DRAINAGE LINE FOR (1) MASTER BATHROOM
- INSTALL NEW WATER SUPPLY FOR (1) MASTER BATHROOM
- INSTALL AIR VENTS PER CODE FOR BATHROOM PLUMBING FIXTURES
- INSTALL SHOWER VALVES AND FIXTURES FOR MASTER BATHROOM
- INSTALL MEMBRANE FOR WATERPROOFING IN BATHROOM SHOWER PAN
- INSTALL THE EXHAUST DUCT THROUGH THE ROOF FOR (1) BATHROOM
- INSULATE HOT WATER PIPES

**9. Insulation**
- INSTALL R-VALUE INSULATION ON THE ADDITION EXTERIOR WALLS PER APPROVED PLANS
- INSTALL R-VALUE INSULATION ON THE ADDITION ATTIC PER APPROVED PLANS
- ANY ADDITIONAL INSULATION DETAIL WILL BE DONE PER APPROVED PLANS

**10. Drywall, Texture and Paint**
- INSTALL DRYWALL THROUGHOUT THE ADDITION AND REMODELED WALLS PER APPROVED PLANS
- TAPE CONNECTIONS
- APPLY TEXTURE TO MATCH WITH EXISTING HOUSE
- APPLY HIGH QUALITY PRIMER
- APPLY TWO COATS OF PAINT WITH COLORS CHOSEN BY CLIENT

**11. Stucco/Siding**
- INSTALL WATERPROOF TAR PAPER ON THE ADDITION
- INSTALL METAL LATH AROUND THE NEW EXTERIOR WALLS PRIOR TO STUCCO INSTALLATION
- APPLY SCRATCH COAT
- APPLY (2) ADDITIONAL COATS OF STUCCO TO MATCH

**12. Flooring**
- INSTALL FLOORING (HARDWOOD/CARPET/LUXURY VINYL) THROUGHOUT THE ADDITION
- INSTALL NEW BASEBOARDS
- APPLY PRIMER AND PAINT NEW BASEBOARDS

**BATHROOM-SPECIFIC STAGES (13-18)**

**13. Shower/Tub Construction**
- INSTALL CURB 4 INCH MIN PER CODE
- INSTALL CONCRETE FOR SHOWER PAN SLOPE PER CODE- 1/4 INCH PER 1 FT
- INSTALL SHAMPOO NICHE
- APPLY THINSET PRIOR TILE INSTALLATION
- INSTALL TILES ON SHOWER WALLS AND SHOWER PAN
- INSTALL MOSAIC TILES ON SHAMPOO NICHE
- APPLY GROUT BETWEEN TILES

**14. Bathroom Floor Tile**
- INSTALL TILES ON THE BATHROOM FLOOR
- APPLY GROUT BETWEEN TILES

**15. Vanity and Fixtures Installation**
- INSTALL DRYWALL ON THE REMAINING WALLS OF THE BATHROOM
- INSTALL NEW VANITY CABINET
- CUT COUNTERTOP TO SHAPE
- INSTALL NEW TOILET
- INSTALL NEW SINK AND FAUCET

**16. Bathroom Electrical (Finish)**
- INSTALL NEW FIXTURE LIGHT

**17. Bathroom Plumbing (Finish)**
- FINAL CONNECTION OF SHOWER VALVE AND FIXTURES
- INSTALL SHOWER HEAD AND TRIM

**18. Bathroom Accessories**
- INSTALL ALL BATHROOM ACCESSORIES (TOWEL BARS, TOILET PAPER HOLDER, GRAB BARS, MIRROR)

### Finish Materials - Master Suite Addition

| Category | Materials |
|----------|-----------|
| Roofing | HD shingles with UV protection, rain gutters, downspouts |
| Exterior | Stucco or siding to match existing |
| Windows | New windows with flashing paper waterproofing |
| Doors | Exterior door (if applicable), interior doors |
| Insulation | R-value insulation (walls and attic) |
| Bedroom Flooring | Hardwood, carpet, or luxury vinyl |
| Baseboards | New baseboards, primed and painted |
| Lighting | Recessed lights throughout |
| Bathroom Flooring | Floor tiles |
| Vanity | Vanity cabinet (single or double sink) |
| Countertop | Stone or solid surface |
| Shower Tile | Wall tiles, floor tiles, mosaic for niche |
| Bathroom Fixtures | Toilet, sink, faucet, shower valve, shower head |
| Bathroom Accessories | Towel bars, toilet paper holder, grab bars, mirror |
| Ventilation | Exhaust fan with humidity sensor |

## Notes for Implementation

### 1. Stage Dependencies

Many stages have dependencies and must be completed in sequence:

- Foundations must be poured before framing can begin
- Framing must be complete before roof installation
- Rough electrical and plumbing must pass inspection before insulation
- Insulation must pass inspection before drywall
- Drywall must be complete before interior paint
- Shower pan must pass inspection before tile installation

### 2. Inspection Milestones

Common inspections for addition projects:

| Inspection | Stage |
|------------|-------|
| Trenches Completed | After excavation for foundations |
| Rebar Inspection | Before pouring concrete |
| Foundation Inspection | After concrete pour |
| Framing Inspection | After framing complete |
| Progress Roofing Inspection | During roofing work |
| Waterproof Inspection | After waterproofing installed |
| Rough Inspection | After electrical/plumbing rough-in |
| Insulation Inspection | After insulation installed |
| Drywall Inspection | After drywall installed |
| Shower Pan Inspection | After shower pan waterproofed |
| Final Inspection | Project completion |

### 3. Code Compliance

All work must comply with:

- California Building Code (CBC)
- Title 24 Energy Requirements
- Local municipal codes and ordinances

Key code requirements:

- GFCI outlets required in wet areas (kitchens, bathrooms) with tamper-resistant features
- Kitchen outlets must be spaced 3 feet apart per code
- Shower curb minimum 4 inches per code
- Shower pan slope 1/4 inch per 1 foot per code
- R-value insulation requirements per Title 24
- Exhaust fans required in all bathrooms
- Smoke and carbon monoxide detectors per code
