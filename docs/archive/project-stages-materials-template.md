## Kitchen Remodel

### Stages

1. **Drawings/Design**
   - Design and drafting of existing and proposed floor plans
   - Design consultation, material selection, 3D rendering
   - Submit to city to obtain permit
   - Revise any changes made by city and resubmit until approval

2. **Demolition**
   - Provide dumpsters as needed
   - Cover all work area as needed prior to demolition
   - Demolish kitchen cabinets, countertops, backsplash, appliances
   - Demolish walls as needed (e.g., pony-wall between kitchen and living room)
   - Demolish windows as needed
   - Clean up and haul all debris

3. **Foundations** (if wall removal requires structural support)
   - Dig to accommodate concrete foundations per approved plans
   - Install rebar per approved plans
   - Pour concrete for new footings per approved plans

4. **Framing** (if layout changes)
   - Install framing for new wall openings
   - Install load-bearing elements to support new open space
   - Install framing for new windows

5. **Electrical**
   - Install electrical outlets per code (3 feet spacing) - tamper resistant GFCI outlets
   - Replace/install ceiling lights (recessed lights)
   - Install undercabinet LED lights
   - Install new electrical connections for kitchen appliances per code
   - Install new switches

6. **Plumbing**
   - Upgrade/relocate plumbing per code for kitchen water supply
   - Install hot and cold valves, hot and cold-water pipes
   - Install sewer, drain and air return vents
   - Upgrade duct for kitchen hood
   - Insulate hot water pipes
   - Relocate kitchen registers (if applicable)

7. **Drywall, Texture and Paint**
   - Install drywall throughout remodeled walls
   - Tape connections
   - Apply texture in remodeled walls to match existing house
   - Apply high quality primer
   - Apply two coats of paint with colors chosen by client

8. **Kitchen Cabinet Installation**
   - Install selected cabinets throughout kitchen and island
   - Install plywood throughout kitchen cabinets prior to countertop installation
   - Install toe kicks and crown molding as desired

9. **Kitchen Countertop Installation**
   - Pick up slabs from vendor, cut in facility, deliver to job site
   - Cut to shape, bullnose as needed
   - Install new countertops on kitchen and island

10. **Kitchen Finish Work**
    - Install backsplash on kitchen walls up to cabinet (stone/mosaic)
    - Install kitchen hood
    - Install new sink and faucet
    - Install garbage disposal
    - Install all appliances (cooktop, microwave, oven, fridge, dishwasher)

### Finish Materials - Kitchen

| Category | Materials |
|----------|-----------|
| Cabinets | Selected cabinets (style per owner selection) |
| Countertops | Stone slabs (granite, quartz, marble) |
| Backsplash | Stone/mosaic tile |
| Flooring | Hardwood, engineered wood, luxury vinyl (SPC) |
| Lighting | Recessed lights (6"), undercabinet LED lights |
| Appliances | Cooktop, microwave, garbage disposal, oven, fridge, dishwasher, hood |
| Plumbing Fixtures | Sink, faucet |
| Trim | Toe kicks, crown molding, baseboards |

### Common Variations

- With or without island
- With or without wall removal (structural changes)
- With or without window replacement
- Addition of sun tunnels
- Vaulted/cathedral ceiling conversion

## Bathroom Remodel

### Stages

1. **Drawings/Design**
   - Design and drafting of existing and proposed floor plans
   - Submit to city to obtain permit
   - Revise any changes made by city and resubmit until approval

2. **Demolition**
   - Provide dumpsters as needed
   - Cover all work area as needed prior to demolition
   - Demolish shower tile, cabinets, countertop, sink, toilet
   - Demolish bathtub (if applicable)
   - Demolish light fixtures and bathroom accessories
   - Clean up and haul all debris

3. **Electrical**
   - Install new vacancy sensor per code
   - Install new outlets and switches - tamper resistant GFCI outlets per code
   - Install new fixture lights
   - Install new humidity sensor for exhaust fan
   - Install recessed lights (typically 2 per bathroom)
   - Install new hard line to panel for exhaust fan

4. **Plumbing**
   - Upgrade plumbing per code
   - Install new shower valve
   - Upgrade vanity plumbing per code
   - Upgrade air vents per code for plumbing fixtures
   - Install/upgrade exhaust fan duct per code
   - Install waterproof membrane for shower pan
   - Insulate hot water pipes

5. **Shower/Tub Construction**
   - Install curb 4 inch min per code
   - Install concrete for shower pan slope per code (1/4 inch per 1 ft)
   - Install shampoo niche
   - Install blocks as necessary prior to grab bars installation
   - Install DensShield boards in shower area
   - Apply thinset prior to tile installation
   - Install tiles on shower walls and shower pan
   - Install mosaic tiles on shampoo niche
   - Apply grout between tiles

6. **Drywall, Texture and Paint**
   - Install drywall on remaining walls of bathroom
   - Tape drywall connections
   - Apply high quality primer
   - Apply texture to match and paint

7. **Floor Tile Installation**
   - Install tiles on bathroom floor
   - Apply grout between tiles

8. **Vanity and Fixtures Installation**
   - Install new vanity cabinet
   - Cut countertop to shape
   - Install backsplash (4-6 inch)
   - Install new toilet
   - Install new sink and faucet
   - Install grab bars (if applicable)
   - Install all bathroom accessories

### Bathroom Types

#### Walk-in Shower
- Install curb 4 inch min per code
- Install concrete for shower pan slope per code
- Install shampoo niche
- Install DensShield boards in shower area
- Install tiles on shower walls and shower pan
- Install mosaic tiles on shampoo niche

#### Bathtub
- Install mortar as necessary prior to tub installation
- Install new tub
- Install shampoo niche
- Install tiles all around bathtub walls

#### Half Bathroom (Powder Room)
- No shower/tub work
- Floor tile installation
- Vanity cabinet or pedestal sink
- Toilet installation
- Light fixture and accessories

### Finish Materials - Bathroom

| Category | Materials |
|----------|-----------|
| Vanity | Vanity cabinet (single or double sink) |
| Countertop | Stone or solid surface, cut to shape |
| Backsplash | 4-6 inch backsplash tile |
| Shower Tile | Wall tiles, floor tiles, mosaic tiles for niche |
| Floor Tile | Bathroom floor tiles |
| Fixtures | Toilet, sink, faucet, shower valve, shower head |
| Accessories | Towel bars, toilet paper holder, grab bars, mirror |
| Lighting | Recessed lights (6"), vanity fixture light |
| Ventilation | Exhaust fan with humidity sensor |
| Shower Features | Shampoo niche, curb, tile shelves |

### Common Variations

- Master bathroom with walk-in shower
- Guest bathroom with bathtub
- Jack and Jill bathroom (shared between bedrooms)
- Freestanding tub installation
- Dual vanity vs single vanity

   - Demolish kitchen cabinets, countertops, backsplash, appliances
   - Demolish windows and doors as needed
   - Demolish flooring throughout house
   - Demolish fireplace/chimney (if applicable)
   - Clean up and haul all debris

3. **Foundations** (if structural changes)
   - Dig to accommodate concrete foundations per approved plans
   - Install rebar per approved plans
   - Pour concrete for foundations per approved plans
   - Install hold downs and epoxy per approved plans
   - Install anchor bolts per approved plans

4. **Framing**
   - Install framing for wall openings
   - Install framing for vaulted/cathedral ceiling (if applicable)
   - Install framing to enclose chimney opening (if applicable)
   - Install load-bearing elements to support new structure
   - Install framing for new windows and doors

5. **Electrical**
   - Install new outlets and switches throughout house - GFCI outlets in wet areas
   - Install recessed lights throughout house
   - Install new hard line to panel for exhaust fans
   - Install new electrical connections for appliances per code
   - Re-wire house per code (if applicable)

6. **Plumbing**
   - Relocate/upgrade drainage lines per code
   - Relocate/upgrade water supply per code
   - Install exhaust fan ducts for bathrooms
   - Install new exhaust duct for kitchen hood
   - Insulate hot water pipes

7. **Drywall, Texture and Paint**
   - Install drywall throughout remodeled walls
   - Tape connections
   - Apply texture to match existing house
   - Apply high quality primer
   - Apply two coats of paint with colors chosen by client

8. **Flooring**
   - Install flooring (luxury vinyl, engineered hardwood) throughout house
   - Install new baseboards throughout house
   - Apply primer and paint new baseboards

9. **Bathroom Remodel** (multiple bathrooms)
   - Follow bathroom remodel stages for each bathroom

10. **Kitchen Remodel**
    - Follow kitchen remodel stages

11. **Windows and Doors**
    - Waterproofing prior to windows/doors installation with flashing paper
    - Installation of window/door frame
    - Installation of windows/doors

12. **Stucco/Exterior** (if applicable)
    - Install waterproof tar paper on remodeled areas
    - Install metal lath around exterior walls
    - Apply scratch coat
    - Apply additional coats of stucco

## Addition (Room Addition)

### Stages

1. **Drawings/Design**
   - Design and drafting of existing and proposed floor plans and elevations
   - Preparation of stamped structural calculations and detailed drawings
   - Analysis for wind, seismic and gravity forces per California Building Code
   - Title 24 energy calculation services
   - Submit to city full set of blueprints
   - Revise any changes made by city and resubmit until approval

2. **Demolition**
   - Provide dumpsters as needed
   - Cover all work area as needed prior to demolition
   - Demolish exterior walls where addition connects
   - Demolish deck/patio/porch as needed
   - Clean up and haul all debris

3. **Foundations**
   - Dig to accommodate concrete foundations per approved plans
   - Install rebar per approved plans
   - Pour concrete for foundations per approved plans
   - Pour concrete for slabs per approved plans (if applicable)
   - Install hold downs and epoxy per approved plans
   - Install anchor bolts per approved plans

4. **Framing**
   - Install framing for addition (sq-ft specified)
   - Extend existing exterior walls to accommodate additional square footage
   - Connect new walls to existing structure for structural integrity
   - Frame interior walls per architectural plans
   - Install load-bearing elements to support new structure
   - Install framing for new floor, ceiling and roof
   - Install framing for new windows and doors

5. **Roof**
   - Install plywood for roofing in addition
   - Install waterproof tar paper on new roof
   - Install metal flashing drip edge and around pipes and vents
   - Install HD shingles with UV protection to match existing
   - Install rain gutters in new roof
   - Install downspouts

6. **Windows and Doors**
   - Waterproofing prior to windows/doors installation with flashing paper
   - Install new windows and doors throughout addition

7. **Electrical**
   - Install new outlets and switches throughout addition
   - Install recessed lights throughout addition
   - Install new hard line to panel for exhaust fans
   - Install electrical connections for appliances per code

8. **Plumbing** (if addition includes bathroom/kitchen)
   - Install new drainage line
   - Install new water supply
   - Install air vents per code
   - Install exhaust duct through roof
   - Insulate hot water pipes

9. **Insulation**
   - Install R-value insulation on addition exterior walls per approved plans
   - Install R-value insulation on addition attic per approved plans

10. **Drywall, Texture and Paint**
    - Install drywall throughout addition and remodeled walls
    - Tape connections
    - Apply texture to match existing house
    - Apply high quality primer
    - Apply two coats of paint with colors chosen by client

11. **Stucco/Siding**
    - Install waterproof tar paper on addition
    - Install metal lath around new exterior walls
    - Apply scratch coat
    - Apply additional coats of stucco to match
    - Or install siding panels to match existing

12. **Flooring**
    - Install flooring (hardwood/carpet/luxury vinyl) throughout addition
    - Install new baseboards
    - Apply primer and paint new baseboards

13. **Interior Finishes** (bathroom, kitchen, etc.)
    - Follow respective room stages

### Common Addition Types

|---------------|------------------------|--------------|
| Single Room Addition | 100-300 sq-ft | Office, bedroom, family room |
| Master Suite Addition | 400-600 sq-ft | Master bedroom, master bathroom, walk-in closet |

## ADU (Accessory Dwelling Unit)

### Stages

1. **Drawings/Design**
   - Provide survey report showing property boundaries and P.U.E limits
   - Design and drafting of existing and proposed floor plans and elevations
   - Preparation of stamped structural calculations and detailed drawings
   - Analysis for wind, seismic and gravity forces per California Building Code
   - Title 24 energy calculation services
   - Submit to city full set of blueprints
   - Revise any changes made by city and resubmit until approval
   - Material selection, space planning, design consultation and 3D rendering

2. **Demolition**
   - Provide dumpsters as needed (trash, concrete/dirt)
   - Cover all work area as needed prior to demolition
   - Demolish hardscape/landscape as needed
   - Clean up and haul all debris

3. **Foundations**
   - Dig to accommodate concrete foundations per approved plans
   - Break concrete as needed for drainage, grading and sewer
   - Install rebar per approved plans
   - Pour concrete foundation per approved plans
   - Install hold downs and epoxy per approved plans
   - Install floor joists, subfloor

4. **Framing**
   - Install framing for ADU (specified sq-ft)
   - Install subfloor and wall framing per plans
   - Install roof trusses framing per plans
   - Frame interior walls per architectural plans
   - Install load-bearing elements

5. **Roof**
   - Install plywood for roofing in ADU
   - Install waterproof tar paper on new roof
   - Install metal flashing drip edge and around pipes and vents
   - Install HD shingles with UV protection for energy efficiency
   - Install rain gutters and downspouts

6. **Windows and Doors**
   - Waterproofing prior to windows/doors installation with flashing paper
   - Install windows and doors throughout ADU

7. **Electrical**
   - Install new ADU main panel or sub panel
   - Install new outlets and switches throughout ADU - GFCI outlets
   - Install recessed lights throughout ADU
   - Install electrical connections for appliances
   - Install electrical connections for water heater
   - Install electrical connections for mini split
   - Install electrical connections for washer and dryer

8. **Plumbing**
   - Install new main drainage line for ADU
   - Install new main water supply for ADU
   - Install drainage and water supply for bathroom, kitchen, laundry
   - Install air vents per code for plumbing fixtures
   - Install membrane for waterproofing in shower pan
   - Install exhaust duct for bathroom and kitchen
   - Insulate hot water pipes

9. **Insulation**
   - Install R-value insulation on ADU floor and exterior walls
   - Install R-value insulation on ADU attic

10. **Drywall, Texture and Paint**
    - Install drywall throughout ADU
    - Tape connections
    - Apply texture
    - Apply high quality primer throughout ADU
    - Apply two coats of paint with colors chosen by client

11. **Flooring**
    - Install flooring (hardwood/carpet/luxury vinyl) throughout ADU
    - Install new baseboards throughout ADU
    - Apply primer and paint new baseboards

12. **Stucco/Siding**
    - Install waterproof tar paper on ADU
    - Install metal lath around exterior walls
    - Apply scratch coat
    - Apply additional coats of stucco
    - Apply primer and paint
    - Or install siding panels to match

13. **Kitchen**
    - Install selected cabinets throughout kitchen
    - Pick up slabs, cut and install countertops
    - Install toe kicks and crown molding
    - Install kitchen hood
    - Install sink and faucet
    - Install backsplash
    - Install garbage disposal
    - Install all appliances

14. **Bathroom**
    - Follow bathroom remodel stages

15. **Additional Work**
    - Install new water heater
    - Install new washer and dryer
    - Install mini split HVAC

## JADU (Junior ADU)

### Description

A Junior ADU is a smaller unit (typically under 500 sq-ft) created within an existing single-family home or attached to it. JADUs are simpler than full ADUs.

### Stages

1. **Drawings/Design**
   - Design and drafting of existing and proposed floor plans and elevations
   - Preparation of stamped structural calculations and detailed drawings
   - Analysis for wind, seismic and gravity forces per California Building Code
   - Title 24 energy calculation services
   - Submit to city full set of blueprints
   - Revise any changes made by city and resubmit until approval

2. **Demolition**
   - Provide dumpsters as needed
   - Cover all work area as needed prior to demolition
   - Demolish walls per approved plans
   - Demolish doors/windows as needed
   - Clean up and haul all debris

3. **Foundations** (if extending footprint)
   - Dig to accommodate concrete foundations
   - Install rebar
   - Pour concrete for foundations and slab
   - Install hold downs and anchor bolts

4. **Framing**
   - Install framing for JADU
   - Extend existing exterior walls
   - Connect new walls to existing structure
   - Install load-bearing elements
   - Install framing for windows and doors

5. **Roof** (if new roof section)
   - Install plywood for roofing
   - Prepare roofing area prior to SPF installation
   - Spray SPF foam throughout roof (or install shingles)
   - Apply finish coat on roof
   - Install rain gutters and downspouts

6. **Windows and Doors**
   - Waterproofing prior to installation with flashing paper
   - Install new windows and doors

7. **Electrical**
   - Install new outlets and switches
   - Install recessed lights
   - Install electrical connections for kitchenette

8. **Plumbing**
   - Install new drainage line for kitchenette
   - Install new water supply for kitchenette
   - Install air vents per code
   - Install exhaust duct for hood/exhaust fan
   - Insulate hot water pipes

9. **Insulation**
   - Install R-value insulation on exterior walls

10. **Drywall, Texture and Paint**
    - Install drywall throughout JADU
    - Tape connections
    - Apply texture to match existing house
    - Apply high quality primer
    - Apply two coats of paint

11. **Flooring**
    - Install flooring (tile) throughout JADU
    - Install new baseboards
    - Apply primer and paint baseboards

12. **Siding**
    - Install waterproof tar paper
    - Cut and install siding panels to match existing house
    - Seal and caulk around all connections
    - Apply primer and paint

13. **Kitchenette**
    - Install fridge
    - Install pre-made kitchenette set (sink and cooktop)

### Finish Materials - JADU

| Category | Materials |
|----------|-----------|
| Flooring | Tile |
| Kitchenette | Pre-made kitchenette set with sink and cooktop, fridge |
| Exterior | Siding to match existing house |
| Windows | New windows with flashing paper |

## Exterior/Stucco Remodel

## Notes for Developers

1. **Stage Dependencies**: Many stages have dependencies. For example, drywall cannot begin until rough inspections pass.

2. **Payment Milestones**: Payment schedules typically align with inspection milestones or stage completions.

3. **Permit Requirements**: All projects requiring structural changes need city permits and inspections.

4. **Code Compliance**: All electrical and plumbing work must be per code (California Building Code, Title 24).

5. **GFCI Outlets**: Required in wet areas (kitchens, bathrooms, laundry) with tamper-resistant features.

6. **Spacing Requirements**: Kitchen outlets must be spaced 3 feet apart per code.

7. **Waterproofing**: Critical for showers (membrane on pan), exterior walls (tar paper), and windows/doors (flashing paper).

8. **Matching Existing**: For remodels and additions, texture, stucco, siding, and roofing should match existing house.

9. **Owner Selections**: Many finish materials require owner selection (colors, styles, materials) before installation.

10. **Warranty**: Standard 5-year labor warranty starting upon passing final inspection.
